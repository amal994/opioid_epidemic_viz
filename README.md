# opioid_epidemic_viz

## Installation & Usage

    git clone https://github.com/amal994/opioid_epidemic_viz
    cd expack
    npm install
    
    npm run buildDev        // for development
        // OR
    npm run buildProd
    
    npm start               // navigate to localhost:8080 for local dev

### For testing

    npm test                // runs test
    npm run coverage        // generates a coverage report
